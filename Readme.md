# Kalimba Tabs Viewer


A simple and elegant Android app for displaying Kalimba tablatures, built with **Jetpack Compose**. The app allows musicians to view Kalimba tabs, consisting of synchronized note and text lines, in a clear and correctly formatted way on their device.

[Play Store](https://play.google.com/store/apps/details?id=com.kontranik.kalimbatabsviewer)

-   **Clear Tab Display**: Renders synchronized note and text lines, just as they are found in traditional tablature files.
-   **Intelligent Line Wrapping**: A custom algorithm ensures that long tablature lines are wrapped cleanly at word boundaries (spaces). This guarantees maximum readability without splitting words or note groups.
-   **Responsive Layout**: The display automatically adapts to the screen width of the device.
-   **Modern UI**: The entire user interface is built with Jetpack Compose, Android's modern UI toolkit.
